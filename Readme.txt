1.	First git pull
2.	Open the backend folder in command panel and type npm init -y
3.	Afterwards type npm i mongodb express cors dotenv in command panel
4.	Now, Open the frontend folder in command panel and type npm install
5. 	Afterwards type npm run dev in command panel
6. 	Open web browser with link http://localhost:5173/ , if not, you can open it manually.
